# Solana Wallet Assistant 🚀

> Submission untuk Colosseum Frontier Hackathon

## Deskripsi
Tool berbasis Python untuk berinteraksi dengan blockchain Solana. Memudahkan pengguna untuk membuat wallet, mengecek saldo, dan melakukan transaksi di Solana DevNet.

## Fitur Utama
- ✅ Generate wallet Solana baru (keypair)
- ✅ Request airdrop SOL di DevNet
- ✅ Cek saldo wallet
- ✅ Integrasi langsung dengan Solana RPC
